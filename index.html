Game Rules

  • You have 60 seconds to complete the game.
  • When you see the picture, type in the German word including the article.
  • Words are not case sensitive.
  • If you answer incorrectly, the same picture will come up again randomly.
  • If you answer correctly, the same picture will not come up during the same game.
